# Иконки для Asustor APK пакета (x86_64)

## Требования к иконкам

Asustor ADM требует иконку в формате PNG:

- **icon.png** - 64x64 или 128x128 пикселей (рекомендуется 128x128)

## Формат

- Формат: PNG
- Цветовое пространство: RGB
- Прозрачность: Поддерживается (альфа-канал)
- Фон: Прозрачный или белый

## Размещение

Иконка должна быть размещена в директории `icons/` пакета:

```
asustor/
├── INFO
├── icons/
│   └── icon.png
└── package/
```

## Создание иконок

1. Создайте базовую иконку в векторном формате (SVG, AI)
2. Экспортируйте в PNG с размером 128x128 пикселей
3. Убедитесь, что иконка хорошо выглядит на светлом и темном фоне
4. Оптимизируйте размер файла (используйте инструменты типа pngquant или optipng)

## Примеры инструментов

- **GIMP** - бесплатный графический редактор
- **Inkscape** - векторный редактор для создания SVG
- **ImageMagick** - конвертация и оптимизация изображений
- **pngquant** - оптимизация PNG файлов

## Временные иконки

До создания финальных иконок можно использовать временные placeholder'ы.
Создайте простую иконку с текстом "IP-CSS" для тестирования пакета.
